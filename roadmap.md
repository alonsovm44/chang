# ROADMAP and Ideas to implement

## for v0.1
1. See command
add `chang see <version>`
this shows the changelog for that version
add `chang see last`
this shows the changelog for the latest version

2. enumerate entries 

3. erase command
`chang erase <version> <entry> <type>`
Example
`chang erase 1.3.12 4 -a`
This erases the entry number 4 on the added section of version 1.3.12
